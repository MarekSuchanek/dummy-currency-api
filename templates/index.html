Dummy currency API

This is dummy currency API for MI-AFP/hw10. It does not provide real exchange rates but some random very near to real ones (3rd May 2018).

Authorization

For authorization, you need a token and use it in requests as Authorization header:

Authorization: token SECRET_TOKEN

Endpoints

For non-existing routes, 404 NOT FOUND is returned.

List currencies

Get list of currencies supported by this service. It provides code (ISO 4217), name, and sign for each currency.

This endpoint requires authorization, otherwise 401 UNAUTHORIZED is returned.

Request

GET /currencies

Response (200 OK)

[
    {
        "code": "USD",
        "name": "U.S. Dollar",
        "sign": "$"
    },
    {
        "code": "EUR",
        "name": "Euro",
        "sign": "€"
    },
    {
        "code": "CZK",
        "name": "Czech Koruna",
        "sign": "Kč"
    },
    {
        "code": "GBP",
        "name": "Pound Sterling",
        "sign": "£"
    },
    {
        "code": "CNY",
        "name": "Chinese Yuan Renminbi",
        "sign": "¥"
    }
]

Get rates for currency

Get exchange rates for given currency. The response consists of requested currency code, timestamp and list of rates for other currencies.

This endpoint requires authorization, otherwise 401 UNAUTHORIZED is returned.

If unsupported currency is requested, 400 BAD REQUEST is returned.

Request

GET /exchange-rates/CURRENCY_CODE

Response (200 OK)

Example for /exchange-rates/USD.

{
    "base": "USD",
    "rates": {
        "CNY": 6.421523658213468,
        "CZK": 21.225549188304978,
        "EUR": 0.8338408269993384,
        "GBP": 0.7338134285779605
    },
    "timestamp": "2018-05-03 15:15:22.585259"
}
